
## Hero-секция для лендинга-портфолио (вайбкодинг)

Заменю плейсхолдер на `src/routes/index.tsx` полноценной Hero-секцией в стиле референса AINOVA: тёмный фон с фиолетово-магентовым градиентным свечением, неоновые акценты (зелёный лайм + фиолетовый), крупная жирная типографика в верхнем регистре.

### Что будет сделано

1. **Дизайн-токены** (`src/styles.css`)
   - Добавлю тёмные семантические токены и переопределю `:root` под тёмную схему (фон near-black, фиолетовое свечение).
   - Новые токены: `--neon-lime`, `--neon-purple`, `--neon-magenta`, `--gradient-hero` (радиальный фиолет→магента→чёрный), `--glow-lime`, `--glow-purple`.
   - Регистрирую их в `@theme inline` (`bg-neon-lime`, `text-neon-purple` и т.д.).

2. **Hero-компонент** (`src/routes/index.tsx`)
   - Полноэкранная секция `min-h-screen` с радиальным градиентным фоном и мягкими blur-«световыми пятнами».
   - Левая колонка (текст):
     - H1: «СОЗДАЮ AI-ПРОДУКТЫ ЧЕРЕЗ ВАЙБКОДИНГ» — крупный, uppercase, tight tracking.
     - Подзаголовок: «Быстро собираю MVP, лендинги и веб-приложения с помощью современных AI-инструментов.»
     - Строка-слоган в стиле референса: «Быстро. Гибко. С душой.»
     - Две CTA-кнопки:
       - «Посмотреть проекты» — заливка `neon-lime`, тёмный текст, свечение.
       - «Связаться» — прозрачная с фиолетовой неоновой обводкой и свечением.
   - Правая колонка (визуал):
     - Сгенерирую 3D-рендер через `imagegen` (premium) — тёмная глянцевая четырёхлучевая звезда/кристалл с фиолетово-магентовыми хроматическими бликами и неоновым курсором внутри, прозрачный фон. Сохраню в `src/assets/hero-vibe.png` и импортирую как ES6.
     - Мелкие декоративные «звёзды» вокруг с лёгким float-анимированием.
   - Индикатор скролла снизу (стрелка вниз с неоновой точкой) как на референсе.

3. **Адаптивность**
   - Mobile: одна колонка, визуал сверху уменьшенный, H1 `text-4xl`, кнопки на всю ширину в стек.
   - Tablet: сохраняется одна колонка, но крупнее.
   - Desktop (`lg:`): 2 колонки 1:1, H1 `text-6xl/7xl`.
   - Проверяю читаемость контраста подзаголовка на градиенте.

4. **Мета-теги** (`src/routes/__root.tsx`)
   - Обновлю title/description/og на осмысленные («Вайбкодинг — AI-продукты, MVP и лендинги»), т.к. дефолтные «Lovable App» запрещены.

### Технические детали

- Стек: TanStack Start + Tailwind v4 (токены в `@theme inline` в `styles.css`).
- Только фронтенд-изменения, без бэкенда.
- Все цвета — через семантические токены, без hardcoded hex в JSX.
- Шрифты: подключу через `<link>` в `__root.tsx` head (например, Inter/Space Grotesk для заголовков), т.к. `@import` URL в v4 запрещён.
- Остальные файлы не трогаю.
